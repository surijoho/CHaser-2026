# Suri-CHaser-2026
ここには、数理情報部CHaser大会に参加するためのプログラム一式が入っています。
まずはこのページを上から順番に読んで、自分のPCで動かせるように準備しましょう！

📥 ダウンロード方法
1. 右側の[Releases]をクリック
2. assets の中にある suri-CHaser-starterkit.zipをクリックしてダウンロード！
※保存したZIPを右クリックして 「すべて展開」 して使ってください。

📦 セット内容
ダウンロードしたフォルダには以下の大事なフォルダ/ファイルが入っています。

「python」：動かすための心臓(消さないで！)　<br>
「sakura」：通信用の部品(消さないで！) <br>
「Sample」：サンプルプログラムのバックアップです。 <br>
「U16-AsahikawaProcon-Server」：サーバー本体(消さないで！) <br>

「●編集（ここにファイルを落とす）.bat」 <br>
「★実行①（サーバ起動）.bat」 <br>
「★実行②♪♪♪（ここにファイルを落とす）.bat」 <br>
「〇選手名等の変更.bat」 <br>
「AnimationTime.ini」：サーバーの設定（消さないで！） <br>
「～～.py」：君が作戦を書くメインのファイルです。 <br>
「ダウンロード先.txt」

🚀　準備ステップ<br>
不要！キットをダウンロードすれば始められます！<br>
※windows以外を使っている方は ●編集（ここにファイルを落とす）.bat が使えません。適宜VSCordなどをダウンロードしてください。

🎮 練習のしかた（動かしてみよう）<br>
まずはサンプルコードをそのまま動かしてみましょう！

1.サーバーを立てる<br>
フォルダ内の 「★実行①（サーバ起動）.bat」 ファイルをダブルクリックして起動します。<br>
※不明な発行元と表示されますが、実行してOKです。<br>
<img width="540" height="432" alt="image" src="https://github.com/user-attachments/assets/2b778cfc-cea8-4f39-b3e2-61fc4f2a751e" />
この画面が出ればOKです。<br>
| COOL/HOT：このゲームの先行/後攻<br>
| TCPユーザー▾：自動くん/ManualClientに切り替えられます。<br>
|　 自動くんは動かずその場にいてくれるプログラムです。<br>
| Map：[...]をクリック　U16-AsahikawaProcon-Server/Maps/ の中から好きなマップを選べます。


2.プログラムを実行する<br>
・サーバーのCOOLの[待機開始]をクリック<br>
※ネットワークのアクセスを許可のような画面が出てきますが、[許可]して続けてください<br>
・フォルダの中で実行したいファイルを「ドラッグ」→「★実行②♪♪♪（ここにファイルを落とす）.bat」の上で離す<br>
<img width="1734" height="413" alt="スクリーンショット 2026-04-05 204747" src="https://github.com/user-attachments/assets/422fbc6e-6640-4fff-85fc-cb68f4ba7861" /> <br>
・黒い画面に c と入力してエンター！<br>
<img width="1734" height="413" alt="スクリーンショット 2026-04-05 213911" src="https://github.com/user-attachments/assets/2f8a0db6-266a-4629-9d98-da3b284eb834" /> <br>
・HOTを[自動くん]に切り替える<br>
<img width="540" height="450" alt="スクリーンショット 2026-04-05 213937" src="https://github.com/user-attachments/assets/5aa3e4fd-7227-475e-aeb3-33371790ceef" /> <br>
・[ゲーム開始]をクリック！

3.確認<br>
サーバー画面でコマが動き出したら成功です！

🛠️ プログラムの書き換え方<br>
フォルダの中で編集したいファイルを「ドラッグ」→「●編集（ここにファイルを落とす）.bat」の上で離す<br>
while(True): の中身を書き換えて、自分だけの最強アルゴリズムを作りましょう！<br>
書き換えたら 「保存（Ctrl + S）」 を忘れずに！

💡 困ったら？<br>
対象のコンピューターによって拒否されたため、接続できませんでした。と表示される→先にサーバーを立てているか/待機開始を押しているか確認してください！

最後まで読んでくれてありがとう！実行できたら画面右上の⭐️を押してください！

（分からないことがあれば、Discordの チャンネルで先輩に聞いてください！）
