# Suri-CHaser-2026
ここには、数理情報部CHaser大会に参加するためのプログラム一式が入っています。
まずはこのページを上から順番に読んで、自分のPCで動かせるように準備しましょう！

📥 ダウンロード方法
1. 右側の[Releases]をクリック
2. assets の中にある suri-CHaser-starterkit.zipをクリックしてダウンロード！
※保存したZIPを右クリックして 「すべて展開」 して使ってください。

📦 セット内容
ダウンロードしたフォルダには以下の大事なフォルダ/ファイルが入っています。

python：動かすための心臓(消さないで！)　<br>
sakura：通信用の部品(消さないで！) <br>
Sample：サンプルプログラムのバックアップです。 <br>
U16-AsahikawaProcon-Server：サーバー本体(消さないで！) <br>

●編集（ここにファイルを落とす）.bat <br>
★実行①（サーバ起動）.bat <br>
★実行②♪♪♪（ここにファイルを落とす）.bat <br>
〇選手名等の変更.bat <br>
AnimationTime.ini：サーバーの設定（消さないで！） <br>
～～.py：君が作戦を書くメインのファイルです。 
ダウンロード先.txt

🚀　準備ステップ<br>
不要！キットをダウンロードすれば始められます！<br>
※windows以外を使っている方は ●編集（ここにファイルを落とす）.bat が使えません。適宜VSCordなどをダウンロードしてください。

🎮 練習のしかた（動かしてみよう）<br>
まずはサンプルコードをそのまま動かしてみましょう！

1.サーバーを立てる<br>
★実行①（サーバ起動）.bat をダブルクリックして起動します。<br>
※不明な発行元と表示されますが、実行してOKです。<br>
<img width="540" height="432" alt="image" src="https://github.com/user-attachments/assets/2b778cfc-cea8-4f39-b3e2-61fc4f2a751e" />
この画面が出ればOKです。<br>

2.プログラムを実行する<br>
フォルダの中で「右クリック」→「ターミナルで開く（またはコマンドプロンプト）」を選択。
黒い画面に python main.py と入力してエンター！
確認
サーバー画面でロボットが動き出したら成功です！
🛠️ プログラムの書き換え方
sakura.exe を起動します。
main.py をサクラエディタにドラッグ＆ドロップして開きます。
while True: の中身を書き換えて、自分だけの最強アルゴリズムを作りましょう！
書き換えたら 「保存（Ctrl + S）」 を忘れずに！
💡 困ったら？
「Pythonが見つかりません」と出る → Pythonのインストール時にPATHにチェックを入れたか確認！
「接続できません」と出る → 先にサーバーを起動しているか確認！
分からないことがあれば、Slack/Discordの #chaser-help チャンネルで先輩に聞いてください！
