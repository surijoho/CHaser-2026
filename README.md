# Suri-CHaser-2026
このリポジトリには、数理情報部CHaser大会に参加するためのプログラム一式が入っています。
まずはこのページを上から順番に読んで、自分のPCで動かせるように準備しましょう！

📥 ダウンロード方法
上のファイル一覧にある CHaser2024_StarterKit.zip をクリック。
右上の [Download] ボタン（または「...」の中のDownload）を押して保存。
保存したZIPを右クリックして 「すべて展開」 して使ってください。

📦 セット内容
ダウンロードしたフォルダには以下の大事なファイル/フォルダが入っています。

python：動かすための心臓(消さないで！)

main.py：君が作戦を書くメインのファイルです。

CHaserServer.exe：練習用の対戦サーバーです。

sakura.py：通信用の大事な部品です（消さないで！）。

🚀 準備ステップ
1. Pythonのインストール
プログラムを動かすために「Python」が必要です。
python.org から最新版をダウンロードしてインストールしてください。
重要： インストール画面で 「Add Python to PATH」 というチェックボックスに必ずチェックを入れてください！
2. このキットのダウンロード
このページの右上にある緑色の [<> Code] ボタンをクリック。
[Download ZIP] を選択してダウンロード。
ダウンロードしたZIPファイルを右クリックして 「すべて展開（解凍）」 してください。
※解凍せずに実行するとエラーになります！

🎮 練習のしかた（動かしてみよう）
まずはサンプルコードをそのまま動かしてみましょう！
サーバーを立てる
フォルダ内の CHaserServer.exe をダブルクリックして起動します。
画面に「待機中」と出ればOKです。
プログラムを実行する
フォルダの中で「右クリック」→「ターミナルで開く（またはコマンドプロンプト）」を選択。
黒い画面に python main.py と入力してエンター！
確認
サーバー画面でロボットが動き出したら成功です！
🛠️ プログラムの書き換え方
sakura.exe を起動します。
main.py をサクラエディタにドラッグ＆ドロップして開きます。
while True: の中身を書き換えて、自分だけの最強アルゴリズムを作りましょう！
書き換えたら 「保存（Ctrl + S）」 を忘れずに！
💡 困ったら？
「Pythonが見つかりません」と出る → Pythonのインストール時にPATHにチェックを入れたか確認！
「接続できません」と出る → 先にサーバーを起動しているか確認！
分からないことがあれば、Slack/Discordの #chaser-help チャンネルで先輩に聞いてください！
